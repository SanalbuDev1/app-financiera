# Módulo de Préstamos y Deudas

## Descripción General

Este módulo permite gestionar deudas personales con cálculo automático de amortización, seguimiento de pagos y alertas de vencimiento.

## Características Principales

1. **Amortización Francesa** - Sistema de cuota fija donde cada pago incluye capital + intereses
2. **Frecuencias de Pago** - Soporte para pagos mensuales y quincenales
3. **Abonos Extraordinarios** - Pagos adicionales que reducen la cuota o el plazo
4. **Tabla de Amortización** - Proyección completa de todas las cuotas
5. **Alertas de Vencimiento** - Notificaciones por email/WhatsApp antes del vencimiento
